# Zomato Bangalore Restaurant Analysis

## Objective
To analyze 41,263 restaurant listings in Bangalore to identify 
patterns in ratings, pricing, cuisine preferences, location trends, 
and the impact of online ordering on restaurant performance.

## Tools Used
- **Python** (pandas, numpy, matplotlib, seaborn) — data cleaning & EDA
- **SQL** (pandasql) — business query analysis
- **Power BI** — interactive dashboard

## Dataset
- Source: Kaggle — Zomato Bangalore Restaurants
- Original size: 51,717 rows, 17 columns
- After cleaning: 41,263 rows, 11 columns

## Key Insights
1. Restaurants offering online ordering rate higher on average 
   (3.72) vs those that don't (3.66)
2. BTM and Koramangala dominate restaurant count — highest 
   competition zones in Bangalore
3. Pub, Cafe and Bar type restaurants get highest average ratings
4. Premium restaurants (above ₹700) rate better (3.99) but the 
   gap vs budget (3.57) is not dramatic — price doesn't guarantee quality
5. Lavelle Road has highest average rating (4.14) despite not 
   being the busiest area — quality over quantity

## Files
- `zomato_cleaned.csv` — cleaned dataset
- `analysis.sql` — 5 business SQL queries with insights
- `zomato_dashboard.pdf` — Power BI dashboard export
